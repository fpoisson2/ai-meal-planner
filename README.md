# AI Meal Planner

Application Flask permettant de planifier les repas de la semaine et de tirer parti de l'API OpenAI pour suggérer des repas.

## Installation

```bash
pip install flask openai python-dotenv
```

## Utilisation

1. Renseigner votre clé OpenAI dans `.env` (voir `.env.example`).
2. Lancer l'application :

```bash
python app.py
```

L'interface est disponible sur `http://localhost:5000`.
